# title:  
skyportfolio  
# description:  
元々使っていたフリーサーバの規約変更により、自分でポートフォリオを作成することになりました。  
動的にコンテンツを変更できるようにし、マルチユーザにより管理・運用をすることを目的としています。  
# Skills:  
フロントエンド：Next.js  
バックエンド：Django  
# Demo:

# How to execute:  
## frontend:  
・「frontend」ディレクトリへ移動  
・下記コマンド実行  
```
npm run dev
```
## backend:
・「backend」ディレクトリへ移動  
・下記コマンド実行  
```
python manage.py runserver
```
